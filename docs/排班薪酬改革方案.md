# 排班与薪酬改革方案

> 提案日期：2026年5月18日

## 一、当前痛点

| 问题 | 现状 |
|------|------|
| 全天班排班不合理 | 9:00-19:00 中间夹2小时死档（13:00-15:00），店长"被占用时间"高达299小时/月，时薪体验差 |
| 晚高峰人手不足 | 19:00-20:30 只有下午半天班1人，客流最高峰反而最缺人 |
| 半天班底薪低于法定最低工资 | 2200元/月，连云港2026年1月起最低工资标准2430元，存在合规风险 |
| 提成平均主义 | 店长与员工平分提成，店长没有额外动力管好门店、冲业绩 |
| 加班缺乏约束 | 排班不合理导致频繁加班，成本失控 |

---

## 二、全天班（店长）改革

### 排班时间优化

| 项目 | 现行 | 改革后 |
|------|------|--------|
| 班次时间 | 9:00-19:00（含2h空档） | **10:00-13:30 / 15:30-20:00** |
| 实际工时 | 8h/天 | 8h/天 |
| 月工时 | 208h/30天 | 208h/30天（不变） |
| 底薪 | 3,600元 | 3,600元（不变） |
| 时薪 | 17.3元 | 17.3元 |

**新班次时间线：**

```
10:00 ─────────────── 13:30 ──[休息2h]── 15:30 ─────────────── 20:00
      ← 上午班在岗 →                    ← 下午班在岗 →
                     店长在岗覆盖午高峰              店长在岗覆盖晚高峰
                     (11:30-13:00)                   (17:00-20:00)
```

**为什么这么排：**
- 10:00到岗后立刻参与午高峰（11:30-13:00），此时段3人全在，接待能力最强
- 13:30-15:30 真正休息2小时，不再"困在店里无所事事"
- 15:30回来覆盖晚高峰（17:00-20:00），此时段店长+下午半天班=2人，**晚高峰从1人变2人**
- 总工时仍是8小时，不增加公司成本

### 休息日规则

| 项目 | 现行 | 改革后 |
|------|------|--------|
| 可选休息日 | 任意一天 | **仅限周一~周四** |
| 放弃休息 | 无明确规定 | 按17.3元 × 8h = **138.4元/天** |
| 累积 | — | **不可累积**到下周或下月 |

**为什么限周一~周四：** 周五六日是客流最大日，店长必须在店。如果店长想多赚，放弃休息日有明确补偿标准。

### 加班规则

| 项目 | 现行 | 改革后 |
|------|------|--------|
| 日工时上限 | 无约束 | **8小时封顶** |
| 超时认定 | — | 超过8小时视为加班，**须向领导说明理由** |
| 加班费率 | — | 17.3元/小时 |
| 宽限期 | 3分钟 | **≤15分钟不视为加班**（按正常8h计） |

**为什么要设宽限期：** 15分钟宽限期避免员工"为了凑加班费故意磨蹭"，正常收尾工作不算加班。

## 三、半天班（员工）改革

### 底薪调整

| 项目 | 现行 | 改革后 |
|------|------|--------|
| 底薪 | 2,200元 | **2,500元**（+300） |
| 月工时 | 187.5h | 187.5h（不变） |
| 时薪 | 11.73元 | **13.33元** |
| 轮班模式 | 早/下午两班倒，365天无休 | 不变 |

### 为什么要调

**法律合规是底线。** 连云港市人力资源和社会保障局规定，从2026年1月1日起执行的最低工资标准为 **2,430元/月**。当前半天班底薪2,200元**低于法定最低工资**，存在以下风险：

- 劳动监察大队抽查发现后责令补发+罚款
- 员工离职后申请劳动仲裁，公司需补足差额+赔偿金
- 影响企业信用，后续招工困难

调整到2,500元后，超出最低标准70元，完全合规。**这不是福利，是法律义务。**

### 公平性分析

即使涨到2,500元，半天班时薪13.33元仍然远低于店长17.3元，层级合理：

| 对比项 | 店长（全天） | 员工（半天） |
|--------|-------------|-------------|
| 月工时 | 208h | 187.5h |
| 底薪 | 3,600 | 2,500 |
| 时薪 | 17.3元 | 13.33元 |
| 月休 | 4天 | 0天 |
| 责任 | 管店+带人+业绩 | 操作执行 |

**成本影响：** 每店每月 +600元（2人 × 300元），三店合计 **+1,800元/月**。

## 四、提成分配改革

### 从平均主义到绩效导向

| 角色 | 现行 | 改革后 | 变化 |
|------|------|--------|------|
| 店长（全天班） | 33.3% | **40%** | +6.7% |
| 半天班员工A | 33.3% | **30%** | -3.3% |
| 半天班员工B | 33.3% | **30%** | -3.3% |

**为什么这么调：**
- 店长多拿提成 → 更有动力管店、冲业绩、带团队，与公司利益一致
- 半天班员工底薪已涨300元，提成微降3.3%作为部分对冲，月到手**实际增加**（见下方算账）
- 提成总比例仍为2%，**公司提成支出不增加**

### 实际算账

假设某店月业绩10万元，提成2% = 2,000元：

| | 店长 | 半天班员工（每人） |
|------|------|------|
| 底薪变化 | 3,600 → 3,600（不变） | 2,200 → 2,500（**+300**） |
| 提成变化 | 667 → 800（**+133**） | 667 → 600（-67） |
| **月合计变化** | **+133** | **+233** |
| **年合计变化** | **+1,596** | **+2,796** |

> 三个人月收入都涨了。公司只多出底薪的1,800元/月（三店合计），提成是业绩出的，不增加固定成本。

## 五、改革成本汇总

| 成本项 | 金额 | 说明 |
|--------|------|------|
| 半天班底薪上调 | +1,800元/月 | 三店 × 2人 × +300元 |
| 全天班底薪 | 不变 | 3,600元维持 |
| 提成支出 | 不变 | 总比例2%不变，仅内部分配调整 |
| **固定成本净增** | **+1,800元/月** | **+21,600元/年** |

## 六、对公司的核心价值

| 维度 | 效果 |
|------|------|
| **法律合规** | 底薪2,500 > 连云港最低工资2,430，消除处罚和仲裁风险 |
| **客流覆盖** | 店长10:00-20:00覆盖午晚两个高峰，晚高峰从1人变2人 |
| **成本可控** | 三店合计固定成本仅增加1,800元/月，提成支出不变 |
| **激励对齐** | 店长提成40%，业绩越好拿得越多，与公司利益绑定 |
| **加班治理** | 8h封顶 + 说明理由制度，杜绝无效加班、控制成本 |
| **员工稳定** | 半天班涨薪300元 + 合规保障，减少流失 |
